The Rock, Paper & Scissors (RPS) Game is a HOPR app which is able to use the HOPR network as a p2p gaming platform. Using a HOPR node
as a "referee", players can send off-chain moves to it, which given a known logic (i.e. rock beats scissors, scissors beat paper, and
paper beats rock), can produce a winner. After each node has send its move, the referee displays the winner of the game, and the moves
made by each player.

In this demo app, we will also show the pros and cons of the HOPR protocol when sending private messages.

## Requirements

- a HOPR cluster, and at least a `3` HOPR nodes with their respective `HTTP_ENDPOINT`, `WS_ENDPOINT` and `API_TOKEN`
